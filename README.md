# Healthcare accessibility in Southwest Finland

**This webmap was created as MAAN7771 course work in 2025 by:**

**Laura Lamminen, Matias Kurri, Josefine Richter, Eemeli Vihantola and Sampo Viljanen.**

The map visualises travel time to health care facilities via different transportation modes. The solution responded to challenge provided by Lounaistieto.

Explore the map: [https://sjvilj.github.io/MAAN7771-WEBMAP-BREZELN/](https://utu-geospatial-labs.github.io/2025-webmap-brezeln/)
